Site Plan for Glorf's Grub Grimoire

Site Name

Glorf's Grub Grimoire

The name represents a magical cookbook created by Glorf, a goblin chef, to share recipes.

Site Purpose

The purpose of this site is to provide a fantasy-themed recipe book that introduces visitors to the culinary creations of Glorf, a goblin chef. It aims to showcase recipes from various fantasy races, including humans, elves, and dwarves, promoting the message of unity through food. The website includes:

  • A home page with a mission statement and world building elements
  • A collection of recipes that users can explore and filter.
  • An about page sharing Glorf's story and background.
  • A reviews page where visitors can share their experiences with the recipes.

Scenarios

  • What inspired Glorf to create this cookbook? - Users can read about Glorf's backstory on the about page.
  • How can I find recipes for elven desserts? - The recipes page allows users to filter recipes by type and style, including elven desserts.
  • How can I tell the world of Glorf's glory? - Any user is able to submit a review that will be displayed!
  • How can I learn more about Goblins? - Any user is able to read about Glorfs story and also find fun elements around the website, like the Goblin word of the month on the homepage.

Color Schema

  • #292727 (Dark Gray): Background color for large sections.
  • #2f4f2f (Dark Green): Used for backgrounds in the header, sections, and other main elements.
  • #9b800a (Golden Brown): Utilized for borders and accent colors.
  • #f8d125 (Bright Yellow): Used for headings and highlighted elements.
  • #e0f7e0 (Light Green): Background for specific sections, such as reviews

Typography

The site will use two fonts to create a fantasy aesthetic while ensuring readability:

  • 'IM Fell English', serif: Used for the body text, providing a classic, readable serif style that fits my fantasy setting
  • 'MedievalSharp', cursive: Applied to headers and titles - less readable but it looks good I think
